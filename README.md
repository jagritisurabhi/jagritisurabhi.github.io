# Rock Paper Scissors Game
Basic game designed in HTML, CSS and JavaScript to play against your computer

---------------------------------------------------------------------------------------------------------------------
## HTML - index.html

>Structuring: <br>
>>A heading for the game - **Rock Paper Scissors**  <br>
A paragraph for other static information like - **Make your move** <br>
A container of buttons stating the choice - Rock/Paper/Scissors <br>
HTML entities or Emojis are used in place of words <br>
A class for buttons - The buttons have to be styled for good UI using CSS <br>
A paragraph for scores - but with a span element since this will use JS to pick up the button counts <br>
