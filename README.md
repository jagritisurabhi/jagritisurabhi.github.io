# Rock Paper Scissors Game
Basic game designed in HTML, CSS and JavaScript to play against your computer


# HTML - index.html

Structuring:
A heading for the game - Rock Paper Scissors
A paragraph for other static information like - Make your move
  A container of buttons stating the choice - Rock/Paper/Scissors. HTML entities or Emojis are used in place of words
  A class for buttons - The buttons have to be styled for good UI using CSS
  A paragraph for scores - but with a span element since this will use JS to pick up the button counts
